PaypalDonate for Unity3D
===========

Provide Paypal donate button

Demo: http://gris.ucoz.ru/UnityModules/PaypalDonate/Web/PaypalDonate.html

Description:

Just add prefab on your scene to use Paypal donate button.
Please note that you have to specify your Paypal donate button ID in the script.
You can also setup button location.
